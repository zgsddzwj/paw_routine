宠物健康与日常手帐 (暂定名：PawRoutine)

核心定位： 帮助宠物主科学管理宠物的日常起居和健康档案，减轻记忆负担。
技术架构：SwiftData (本地数据存储) + CloudKit (通过用户的 iCloud 账号实现多设备同步) + UserNotifications (本地离线推送)。
整体风格：采用iOS26 Liquid Glass（液态玻璃）风格
1. App 整体架构与导航
采用底部三 Tab 导航：今日 (Today)、档案 (Profiles)、统计 (Insights)。右侧是一个独立圆形玻璃“+”号按钮用于快速记录。
2. 详细页面功能描述
📍 核心交互：快速记录 (Quick Add Bottom Sheet)
• 点击右侧独立悬浮的“+”号，从底部弹出一个半屏面板。
• 提供大图标快捷操作：🍖 喂食、💧 换水、🦮 遛狗、💊 喂药、💩 排便。
• 点击任意图标，直接记录当前时间。长按图标可修改具体时间或添加备注（如：“今天拉肚子了”）。
📍 Tab 1: 今日看板 (Today)
• 多宠物切换： 顶部有宠物头像的横向滑动列表。选中哪个头像，下方就显示该宠物的数据。
• 每日进度环 (Daily Rings)： 类似 Apple Watch 的圆环设计，展示今天的核心任务完成度（如：遛狗 2 次、喂食 3 次）。
• 时间轴 (Timeline)： 垂直展示今天已经完成的记录，以及即将到来的提醒（如：“下午 6:00 喂晚饭”）。
📍 Tab 2: 宠物档案 (Profiles)
• 基础信息卡片： 宠物头像、名字、品种、性别、绝育状态。
• 年龄计算器： 输入出生日期，自动计算精确到天的年龄，并换算成“人类年龄”。
• 医疗与备忘录区：
  • 疫苗/驱虫记录： 记录上次驱虫时间，并设置下次提醒（利用本地推送，如 30 天后提醒体内驱虫）。
  • 体重追踪： 记录每次称重的体重，点击可查看简单的折线图。
  • 证件夹： 允许用户拍照存储疫苗本或绝育证明，纯本地保存。
📍 Tab 3: 统计与回顾 (Insights)
• 日历视图： 选定一个月，有记录的日子会显示小圆点。
• 习惯分析： 简单的柱状图展示本周遛狗总时长、排便异常次数等。这能给主人极大的成就感。
📍 设置页 (Settings)
• 提醒默认值： 设置常规的早晚喂食时间、遛狗时间，App 会自动生成本地推送。
• 数据导出： 支持将记录导出为 CSV 文件（方便发给兽医看）。
• Pro 版解锁 (IAP)： 免费版支持 1 只宠物，Pro 版买断解锁无限宠物数量、高级统计图表和数据导出功能。
